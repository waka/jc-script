# JC Script

This is the script that add power in javascript by extends and super grammer.

## Usage

Compile into JavaScript.

    jc --compile /path/to/script.jc // output to path/to/script.js

Show other options.

    jc --help

Run sample code.

    jc -c test/extends.jc
